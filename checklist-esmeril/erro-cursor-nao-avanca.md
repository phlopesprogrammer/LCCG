# 🛠️ Erro: Cursor não avança ao digitar espessuras

## 📌 Descrição
Ao preencher os campos de espessura, o cursor permanece no mesmo campo.

## 🎯 Comportamento esperado
O cursor deve avançar automaticamente para o próximo campo após o preenchimento.

## 🔁 Passos para reproduzir
1. Acessar tela de Check List Esmeril
2. Localizar seção de espessuras
3. Preencher um campo

## ❌ Resultado atual
O cursor permanece no mesmo campo.

## 🧠 Observações
- Deve funcionar para todos os campos
- Não pode perder valores digitados
- Considerar apenas dois dígitos após a vírgula
